# grimace
gulp + webpack + FB GIF發佈

demo: [totofish.github.io/grimace](https://totofish.github.io/grimace/)

website: [鬼臉3連拍](https://fb.webgene.com.tw/grimace/)

##Installation
```
$ npm install
$ bower install

$ npm run dev
$ npm run build
$ npm run server
```

###NOTE:
特別處理webpack打包後IE8執行錯誤的問題，另外Facebook分享GIF圖片需在有SSL安全協議網站下，否則無法分享。
